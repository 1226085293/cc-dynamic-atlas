# mk_nodes

cocos creator 3.x 插件开发模板

## # 初始化

-   将插件目录移动至 **项目目录/extensions_dev** 下
-   使用 vscode 全局替换 mk_nodes 为你的插件名

## # 编译

-   安装编译 npm 包（已安装跳过）
    > npm i @muzzik/cc-plugin-cli
-   使用
    > cc-plugin-cli c -p '插件绝对目录'
